# Nimbus Hono

Adapters and useful functionality to bridge Nimbus and [Hono](https://hono.dev/).

[https://jsr.io/@nimbus/oak](https://jsr.io/@nimbus/hono)

### Deno

```bash
deno add jsr:@nimbus/hono
```

### NPM

```bash
npx jsr add @nimbus/hono
```

### Bun

```bash
bunx jsr add @nimbus/hono
```
