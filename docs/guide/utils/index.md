# Nimbus Utils Package

The Utils package provides various utility functions.

[https://jsr.io/@nimbus/utils](https://jsr.io/@nimbus/utils)

### Deno

```bash
deno add jsr:@nimbus/utils
```

### NPM

```bash
npx jsr add @nimbus/utils
```

### Bun

```bash
bunx jsr add @nimbus/utils
```
